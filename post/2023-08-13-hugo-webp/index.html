hugo 发布时将图片转成 webp 以及缩小图片质量

2023, 八月 13 · 少侠 | ✍️
目录
  • 将 layouts/_default/_markup/render-image.html 内容替换成下面的 html
{{- $url := urls.Parse .Destination -}}
{{- $scheme := $url.Scheme -}}

{{ $img := .Page.Resources.GetMatch $url.Path }}

<!-- 如果页面中没有找到，则去找post文件夹的资源，必须放一个 _index.md 在 post 文件夹中 -->
{{ if not $img }}
    {{ with .Page.Site.GetPage "section" "post" }}
    {{ $resources := index (.Resources.Match $url.Path) 0 }}
	    {{ with $resources }}
	        {{ $img = $resources }}
	    {{ end }}
    {{ end }}
{{end}}

{{ with $img }}
    {{ if eq $img.MediaType.SubType "gif" }} 
        {{ $img = $img.Resize (printf "%dx gif q60" (int (div $img.Width 1.6) )) }}
    {{ else }}
        {{ $img = $img.Resize (printf "%dx%d webp q80" $img.Width $img.Height) }}
    {{ end }}
{{ end }}

<a target="_blank" href="{{- printf "/%s%s" .Page.File.Dir .Destination | safeURL -}}">
<img alt="图片加载失败" title="点击查看原图" data-fancybox="gallery" data-caption="放大图片" loading="lazy" src="
  {{- if eq $scheme "" -}}
    {{- if strings.HasSuffix $url.Path ".md" -}}
      {{- relref .Page .Destination | safeURL -}}
    {{- else -}}
        {{ if $img }}
            {{ $img.RelPermalink }}
        {{ else }}
            {{- printf "/%s%s" .Page.File.Dir .Destination | safeURL -}}
        {{ end }}
    {{- end -}}
  {{- else -}}
    {{- .Destination | safeURL -}}
  {{- end -}}"
  {{- with .Title }} title="{{ . | safeHTML }}"{{- end -}}
  {{- with .Text }} alt="{{ . | safeHTML }}"
  {{- end -}}
/></a>
{{- /* whitespace stripped here to avoid trailing newline in rendered result caused by file EOL */ -}}
  • 在 post 文件夹下面放一个 _index.md 文件，内容为：
---
title: Posts
headless: true
---

效果如下：

  1. png、jpg 等格式图片会转成 webp 格式图片并且质量压缩 80%
  2. gif 图片尺寸缩小到62%左右，并保持动态

经过测试，图片压缩比例能达到 80% 以上，之前上传到 github page 上面的图片都比较大。

下面这个图片进行测试：

这张图片压缩之后仅有 233KB，原图可以点击查看，原图大小是 6MB

对 image 进行渲染的代码也是比较简单的，如果是一个 page bundle，那么取 page resources，如果不是，那么取 branch bundle 中的 post 文件夹下面的图片，匹配后进行转换格式与渲染

hugo 在给 gif 图片转码成 webp 的时候还不太完善，会失去 gif 的动态效果，所以我这里 gif 就不转格式了，只缩了大小
