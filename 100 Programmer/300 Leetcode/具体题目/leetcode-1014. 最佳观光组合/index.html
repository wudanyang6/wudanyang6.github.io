题目

1014. 最佳观光组合

解法

转换一下公式

vn 代表 i 与 j 的值
fn 为最大值

因为 values[j] - j 是固定的，所以，只需要求 values[i] + i 的最大值就可以了

vn= values [i]+i+ values [j]−jfn=max( values [i]+i+ values [j]−j
class Solution {
    
    /**
     * @param Integer[] $values
     * @return Integer
     */
    function maxScoreSightseeingPair($values) {
        $len = count($values);
        $max = $maxI = 0;
        for ($i = 0; $i < $len; $i++) {
            $max = max($max, $maxI + $values[$i] - $i);
            $maxI = max($values[$i], $maxI);
        }
        
        return $max;
    }
}
